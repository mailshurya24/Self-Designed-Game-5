Variables are as follows:


player, playerImg;
bullet1Group,bullet2Group;
backgroundImg;
edges;
enemy1,enemy2,enemy3,enemy4,enemy5,enemy6,enemy7,enemy8,enemy9,enemy10,enemyImg,enemy_array=[];
playerBullet1Img,playerBullet2Img;
bossBulletImg,bossArray;
score = 0; 


What they do:


player = main character sprite
playerImg = player Image

bullet1Group = group for player bullet 1 
bullet2Group = group for player bullet 2 

bossArray = array for the bosses

backgroundImg = background Image

edges = creates Edge sprites

enemy1 = enemy1 sprite
enemy2 = enemy2 sprite
enemy3 = enemy3 sprite
enemy4 = enemy4 sprite
enemy5 = enemy5 sprite
enemy6 = enemy6 sprite
enemy7 = enemy7 sprite
enemy8 = enemy8 sprite
enemy9 = enemy9 sprite
enemy10 = enemy10 sprite
enemy11 = enemy11 sprite
enemyImg = enemy Image
enemyArray = array for the enemies

playerBullet1Img = image for the bullet type 1 for the player
playerBullet2Img = image for the bullet type 2 for the player

bossBulletImg = image for the bosses' bullet

score = stores the player's score, initialized with 0
